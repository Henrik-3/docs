# Premium Features

Premium features unlock higher rate limits and additional capabilities for the HenrikDev API.

## Tiers

<CardGroup cols={2}>
  <Card title="Free" icon="circle-dollar-sign">
    Default tier for every API key.

    - Standard key rate limits
    - 300s cache TTL
    - No webhook updates
    - No tracked users
  </Card>

  <Card title="Tier 1" icon="zap">
    US\$10.99/month

    - Up to **130 req/min**
    - **60% reduced caching** — 2-minute caches instead of the default 5-minute caches
    - Webhooks for **10 users**
    - Webhook deliveries within **60 seconds** max
  </Card>

  <Card title="Tier 2" icon="zap">
    US\$15.99/month

    - Up to **200 req/min**
    - **80% reduced caching** — 60-second caches
    - Webhooks for **20 users**
    - Webhook deliveries within **40 seconds** max
  </Card>

  <Card title="Tier 3" icon="crown">
    US\$25.99/month

    - Up to **300 req/min**
    - **90–100% reduced caching** — 30-second caches by default, force refresh for zero cache
    - Webhooks for **50 users**
  </Card>
</CardGroup>
