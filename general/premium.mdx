# Premium Features

Premium features unlock higher rate limits and additional capabilities for the HenrikDev API.

## Tiers

| Tier | Rate limits | Cache TTL | Webhooks | Webhook users |
| --- | --- | --- | --- | --- |
| Free | Standard key limits apply | 300s | Not available | 0 |
| Premium | Higher plan limits when more generous than your key | Plan dependent | Match & MMR updates | Plan dependent |

## Advantages of going premium

- **Higher rate limits** — premium plans can raise the requests allowed per interval above the default key limits. The API automatically uses the more generous limit between your key and your active plan.
- **Fresher cached data** — premium plans may use a shorter cache TTL, so responses update sooner.
- **Webhook updates** — get automatic push notifications when a tracked player finishes a match or their MMR changes.
- **More tracked users** — premium plans include a higher webhook user quota.
- **Faster webhook polling** — premium plans may poll tracked users more frequently.
