# Premium Features

Premium features unlock higher rate limits and additional capabilities for the HenrikDev API.

## Tiers

<CardGroup cols={2}>
  <Card title="Free" icon="circle-dollar-sign">
    Default tier for every API key.

    - Standard key rate limits
    - 300s cache TTL
    - No webhook updates
    - No tracked users
  </Card>

  <Card title="Premium" icon="crown">
    Subscribed via the Discord app store. Plan values depend on the chosen tier.

    - Higher rate limits when the plan is more generous than your key
    - Plan-dependent cache TTL (often shorter for fresher data)
    - Webhook updates for match and MMR changes
    - Higher tracked-user quota
    - Faster webhook polling interval
  </Card>
</CardGroup>
