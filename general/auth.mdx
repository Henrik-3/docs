---

This API requires an API Key. You can generate one by follow the instructions:

- Join on the discord: https://discord.com/invite/X3GaVkX2YN
- Verify
- Go to the **#get-a-key** channel
- Select from the dropdown list **"VALORANT"**
- After that, you can select from the following key types:
  - **"Basic" **⇒ 30req/min, granted without approval from our side
  - **"Advanced"** ⇒ 90req/min, requires our approval
