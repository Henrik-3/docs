# Accounts

<Info>
  Use v2 for console support
</Info>

With these endpoints, you can get basic account data like account level, name, tag, puuid and more

  

## Get Account by Name and Tag (v1)

<CodeGroup>

```bash
curl "https://api.henrikdev.xyz/valorant/v1/account/{name}/{tag}" \
  -H "Authorization: YOUR_API_KEY"
```

```python
import requests

headers = {"Authorization": "YOUR_API_KEY"}
response = requests.get(
    "https://api.henrikdev.xyz/valorant/v1/account/{name}/{tag}",
    headers=headers
)
print(response.json())
```

```javascript
const response = await fetch(
  "https://api.henrikdev.xyz/valorant/v1/account/{name}/{tag}",
  { headers: { Authorization: "YOUR_API_KEY" } }
);
const data = await response.json();
console.log(data);
```

</CodeGroup>

```json
GET /valorant/v1/account/{name}/{tag}
```

## Get Account by Name and Tag (v2)

```json
GET /valorant/v2/account/{name}/{tag}
```

## Get Account by PUUID (v1)

```json
GET /valorant/v1/by-puuid/account/{puuid}
```

## Get Account by PUUID (v2)

```json
GET /valorant/v2/by-puuid/account/{puuid}
```

<Info>
  The v2 endpoints support console platforms and return a `platforms` array in the response.
</Info>
