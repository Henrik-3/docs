# Match

<Info>
  We encourage you to use v4 for a more streamlined data layout and to be on the latest version when v2 will be deprecated sometime in the future
</Info>

Get data for a single match.

 

## Get Match (v2)

```json
GET /valorant/v2/match/{matchid}
```

## Get Match (v4)

```json
GET /valorant/v4/match/{region}/{matchid}
```
