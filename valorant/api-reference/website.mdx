# Website

Get all news articles from the [https://playvalorant.com/news](https://playvalorant.com/news) page.

## Get Website News

```json
GET /valorant/v1/website/{countrycode}
```

### Query Parameters

| Parameter | Type | Description |
| --- | --- | --- |
| `category` | string | Filter by category (e.g. game\_updates, dev, esports, announcements) |
