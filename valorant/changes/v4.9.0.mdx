# v4.9.0

## For external developers

- **Premium webhook user management**
  - Manage tracked accounts through `/public/v1/premium/webhook` and `/public/v1/premium/webhook/users`.
  - Premium webhook endpoints now accept an approved API key in addition to JWT authentication.
  - Configure the match and MMR history response schemas used in webhook payloads.
- **Premium access**
  - An approved Premium API key is created automatically when an entitlement becomes active and the user does not already have an API key.
  - Canceled subscriptions remain usable until the current billing period ends.
- **Premier lookups**
  - Lookups by team name or UUID can resolve live data when a team is not already cached.
  - Name lookups support the optional `affinity` filter.
  - If a name and tag match multiple teams, the API now returns an explicit conflict response so the request can be narrowed.
- **History and statistics**
  - Request history and detailed statistics continue to provide status, timing, endpoint, and bucket information with improved reliability.
  - No integration changes are required for existing clients.
- **API documentation**
  - OpenAPI endpoint descriptions now include clearer summaries for Valorant and public routes.
